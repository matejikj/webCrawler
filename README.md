# Instalation

## For running server

- in directory server do:

```
npm install
```

```
node server.js
```

Server running on localhost:8083

## Client setup

- in directory client do:

```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
